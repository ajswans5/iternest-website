# IterNest Website

Canonical repository for the public IterNest company website and journal.

## Purpose

This repository will contain the public-facing IterNest website, including the homepage, product pages, company pages, and the build journal.

## Start here

Friday should read [`FRIDAY_START_HERE.md`](./FRIDAY_START_HERE.md) before implementation.

## Working structure

```text
content/
  journal/        Source copy for journal posts
  pages/          Source copy for permanent website pages
public/
  assets/
    images/       Logos, screenshots, and site imagery
    icons/        Favicons and interface icons
src/
  components/     Reusable website components
  pages/          Website page implementations
  styles/         Shared styles and design tokens
docs/             Product, content, and deployment notes
archive/          Retired drafts and prior versions
```

## Brand direction

Warm, calm, clear, hopeful, mobile-first, and uncluttered. IterNest should feel human and thoughtful—not corporate or noisy.
